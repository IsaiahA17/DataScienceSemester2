# Data Science Portfolio - Semester 2
#### Submission Date: 17/04/2026
#### Author: Isaiah Andres
This repository demonstrates the usage of five machine learning models, a K-Nearest Neighbour model to try and find out what kind of Kickstarter projects are likely to be the most successful based on their category and other given features, a Support Vector Machine to detect fraudulent credit card transactions, a Convolutional Neural Network to detect different severities of dementia and classify patients without dementia and those with very mild dementia, Recurrent Neural Networks to predict the closing price of a given stock such as Nvidia, and a Fuzzy C Means model to show the difficulty of classifying the type of land mines detected by a sensor based on voltage, sensor height and soil.
