# Data Science Portfolio
This repository demonstrates the usage of five machine learning models, K-Nearest Neighbours to try and find out what kind of Kickstarter projects are likely to be the most successful based on their category and other features,
Support Vector Machines to detect fraudulent credit card transactions, Convolutional Neural Networks to detect different severities of dementia, Recurrent Neural Networks to predict the closing price of a given stock such as Nvidia,
and a Fuzzy C Means model to show the difficulty of classifying the type of land mines detected by a sensor based on voltage, sensor height and soil type. 

### This portfolio is currently a work in progress and will be used for a preliminary submission, I intend to overhaul the write ups for each portfolio to add more detail and also make them look tidier and more readable and potentially use a K-means Cluster model
